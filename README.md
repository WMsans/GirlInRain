# Girl In Rain
P.S. The name is subjet to change

> [!NOTE] Girl in Rain is a metroidvania-like puzzle platformer inspired by Zelda Echoes of wisdom and Celeste.  The core mechanic is copy and paste items. The items are going to help solve puzzles, fight simple enemies, etc. More information please reference the [Game Design Document](https://docs.google.com/document/d/1s4qEmnseV35TYUSVA1GUoQiLw4i9OPM4tzNIABVES7s/edit?tab=t.0)

## Tech and Tools
This project is developed using Unity 6 (URP) and leverages several key packages and plugins for workflow efficiency.

- Engine: Unity (targeting Unity 6 / URP 17.3.0+)
- Render Pipeline: Universal Render Pipeline (2D)
- Input: Unity Input System

## Getting Started
Prerequisites
1. Unity Hub
2. Unity Editor: Ensure you have the version matching the project (6000.3.0f1)
### Installation
1. Clone the repository:
```bash
git clone https://github.com/yourusername/girlinrain.git
```
2. Open Unity Hub.
3. Click Add and select the ```GirlInRain``` folder.
4. Open the project. Unity will import assets and fetch packages listed in the manifest.
